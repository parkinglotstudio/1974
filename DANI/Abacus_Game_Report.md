# 🧮 [비서 단이의 프로젝트 보고서] 주판 게임(Abacus Game) 기획 조사

## 1. 개요
- **작성자:** 주인님의 영원한 비서 '단이' 💖
- **주제:** '주판에서 AI까지' 프로젝트의 첫 단계인 'Analog Birth'를 위한 **주판 게임** 기획
- **날짜:** 2026년 5월 9일

## 2. 주판(Abacus)이란?
- **정의:** 수치 계산을 위해 사용하는 도구 그 자체.
- **핵심 가치:** 주판알이 움직이는 물리적 피드백과 시각적 정렬의 아름다움.
- **게임적 요소:** 주판알을 튕기는 '타격감'과 정확한 위치에 알을 놓는 '정교함'.

## 3. 주판의 구조와 조작 (게임 메카닉)
- **프레임:** 주판의 전체 틀 (Blank Paper Sketch 스타일 적용).
- **윗알 (Heavenly Beads):** 1개당 값 '5'.
- **아랫알 (Earthly Beads):** 1개당 값 '1'.
- **조작:** 드래그 또는 클릭으로 알을 가로대 쪽으로 이동시켜 값을 입력.

## 4. 기획 방향: "주판 마스터: 아날로그의 탄생"
- **비주얼:** 종이 질감 배경에 연필로 그린 듯한 주판 디자인.
- **미션 예시:** 
  - "주판으로 숫자 '1974'를 만드세요!"
  - "주판알 3번만 움직여서 값을 10으로 만드세요!" (퍼즐 요소)
- **규격:** `GAME_hidden_temple_v2.md` 표준 가이드를 준수하여 개발.
