# 🏛️ 1974 GAME PROJECT MASTER GUIDE (MUST READ FIRST)

본 문서는 '1974' 프로젝트의 핵심 철학, 작업 원칙, 그리고 지식 자산 활용법을 총망라한 **최상위 지침서**입니다. AI 개발자는 새로운 세션을 시작하거나 새로운 작업을 진행할 때 **반드시 이 문서를 먼저 읽고 숙지**해야 합니다.

## 🎯 프로젝트 핵심 철학 (Core Philosophy)
1. **메인 슬로건**: "인간은 태어나지만, 플레이어는 첫 코인을 넣는 순간 탄생한다."
2. **서사 구조**: 1974년생 플레이어가 디지털 문명과 함께 성장하는 서사적 노스탤지어 게임.
3. **시각 정체성**: 고감도 1비트 픽셀 아트, 12 FPS 저프레임 감성, CRT 아날로그 노이즈.

---

## 📋 세션 시작 체크리스트 (Pre-flight Checklist)
새로운 대화나 작업을 시작할 때 AI는 다음 순서로 문서를 학습한다:
1. **[1974_GAME.md]**(본 문서)를 읽어 프로젝트의 현재 방향성을 복구한다.
2. 진행할 작업의 성격에 따라 아래 **[지식 자산 목록]** 중 해당 파일을 로드한다.
3. UI 이식 작업 시 **[DEV_PROCESS_IMPROVEMENT.md]**의 5대 원칙을 복기한다.
