# **Digital Evolution Avatar: 시대별 디자인 디렉션 가이드**

본 문서는 기획서 \<주판에서 AI까지 \- 1974 범띠\>의 핵심 콘텐츠인 **디지털 진화 아바타(Digital Evolution Avatar)**의 각 단계별 아트 디자인 체크리스트입니다. 노션(Notion) 등 기획 문서 도구에 바로 복사 및 붙여넣기하여 사용할 수 있도록 구조화된 표 형태로 작성되었습니다.

## **01\. Analog Birth (디지털 이전의 시작)**

| 항목 | 가이드라인 및 예시 |
| :---- | :---- |
| **1\. Art Style** | Hand Drawn, 아동용 크레용 일러스트, 펜 스케치 |
| **2\. Visual Language** | Physical media, 골목길 놀이, 아날로그 유년기 |
| **3\. Silhouette** | 어린아이의 불완전한 인체 비례, 낙서 형태(doodle) |
| **4\. Rendering Style** | 거친 연필 선, 불규칙한 색연필 채색 |
| **5\. Color Palette** | 빛바랜 공책 종이색(Warm white/Yellowish), 흑연색, 탁한 원색 |
| **6\. Typography** | 삐뚤빼뚤한 손글씨(Handwriting) |
| **7\. UI Frame / Shell** | 찢어진 종이 프레임, 국민학교 노트 여백 |
| **8\. Composition** | 노트 중앙에 덩그러니 그려진 형태 (Centered doodle) |
| **9\. Asset Layering** | 종이 배경 \+ 스케치 베이스 \+ 크레용 옷 \+ 참잘했어요 도장 |
| **10\. Iconography** | 몽당연필, 지우개 가루, 딱지, 달고나 국자 |
| **11\. Texture Language** | 종이 질감(Paper grain), 흑연 번짐, 크레용 뭉침 |
| **12\. Motion Language** | 정지 화면이거나 2프레임 스톱모션 애니메이션 (Stop-motion) |
| **13\. Perspective** | 2D 평면 정면도 (Flat 2D) |
| **14\. Mood / Tone** | Nostalgic, 따뜻함, 순수함 |
| **15\. Design Keywords** | hand-drawn sketch, 70s kids, paper texture, analog nostalgia |
